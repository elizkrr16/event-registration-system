# Event Registration System

This project is a web application for registering participants for events.  
It is being developed as part of a graduation project.

## Main features

- Event publishing
- Participant registration
- Participant management
- Attendance tracking
- Event statistics

## Technologies

- React (frontend)
- Node.js + Express (backend)
- MySQL (database)
- REST API

## Repository structure

`client` — frontend application  
`server` — backend API  
`database` — SQL schema for the database
