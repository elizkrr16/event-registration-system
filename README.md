# НаВстречу

## Описание проекта

«НаВстречу» — дипломный full-stack проект по теме:

`Разработка веб-сайта для регистрации участников на мероприятия`

Система предназначена для публикации мероприятий, записи участников, просмотра личных регистраций и управления событиями со стороны администратора.

## Цель проекта

Создать понятное локально запускаемое веб-приложение на React + PHP + MySQL для регистрации участников на мероприятия в учебной среде XAMPP.

## Основные функции

- главная страница с ближайшими мероприятиями
- каталог мероприятий с поиском и фильтрами
- детальная страница мероприятия
- регистрация и вход по email и паролю
- личный кабинет участника
- запись и отмена записи на мероприятие
- административная панель
- создание, редактирование и скрытие мероприятий
- просмотр участников и базовой статистики

## Технологии

- frontend: React + JavaScript + Vite
- backend: PHP 8 + PDO
- database: MySQL
- локальная среда: XAMPP
- взаимодействие: REST-like JSON API
- аутентификация: PHP Session Auth

## Архитектура

## База данных

Используемая БД:
- `event_registration_system`

Таблицы:
- `students`
- `organizers`
- `categories`
- `events`
- `registrations`
- `attendance`

Ключевые особенности схемы:
- в `events` есть `format`, `starts_at`, `ends_at`, `program`, `status`
- в `registrations` есть `cancelled_at`
- в `attendance` хранится отметка посещения
- в `registrations` есть `UNIQUE(event_id, student_id)`
- все связи оформлены внешними ключами

## Запуск backend через XAMPP

1. Запустите `Apache` и `MySQL` в XAMPP.
2. Скопируйте папку `server` в:

```text
C:\xampp\htdocs\event-registration-system\server
```

3. Убедитесь, что API доступен по адресу:

```text
http://localhost/event-registration-system/server/api
```

## Импорт базы данных

Импортируйте файлы по порядку:

1. `database/schema.sql`
2. `database/seed.sql`

Через phpMyAdmin или консоль XAMPP MySQL:

```bash
C:\xampp\mysql\bin\mysql.exe -u root < database\schema.sql
C:\xampp\mysql\bin\mysql.exe -u root < database\seed.sql
```

Стандартные параметры подключения в `server/config/db.php`:
- host: `127.0.0.1`
- port: `3306`
- user: `root`
- password: пустой
- db: `event_registration_system`

## Запуск frontend

Перейдите в папку `client`:

```bash
npm install
npm start
```

Frontend будет доступен по адресу:

```text
http://127.0.0.1:5173
```

Если нужно явно задать адрес API:

```text
VITE_API_BASE_URL=http://localhost/event-registration-system/server/api
```

## Как работает авторизация

В проекте используется `PHP session auth`:

- при входе или регистрации backend записывает пользователя в `$_SESSION`
- frontend отправляет запросы с `credentials: 'include'`
- backend проверяет роль через session и ограничивает доступ к админским эндпоинтам

Это выбранный вариант, потому что он самый простой и устойчивый для локального XAMPP.
